TALLER INTELIGENCIA COMPUTACIONAL

Este repositorio contiene todo lo necesario para ejecutar el taller de Inteligencia Computacional.
El código original ha sido adaptado, ya que el dataset completo y algunos archivos de gran tamaño no se pueden subir a GitHub debido a las restricciones de tamaño.

Para ejecutar el proyecto, el código está preparado para descargar automáticamente el dataset y generar el modelo entrenado (model_yoga.pth).
